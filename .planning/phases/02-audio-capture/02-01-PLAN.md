## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User -> Mikrofon-Permission | Nutzer muss TCC-Permission erteilen bevor Audio-Capture funktioniert |
| Audio Render Thread -> Main Thread | installTap-Callback liefert RMS-Werte die auf MainActor konsumiert werden |
| Defaults -> AudioController | Nutzer-konfigurierte silenceDuration und selectedMicUID werden zur Runtime gelesen |
