<objective>
Sanitize the resampled audio samples array in TranscriptionService before passing to the backend, replacing any non-finite values (NaN, Inf, -Inf) with 0.0.

Purpose: AVAudioEngine occasionally drops render cycles under load (`IOWorkLoop: skipping cycle due to overload`), producing NaN values in the accumulated float buffer. These propagate through AVAudioConverter resampling unchanged, then cause FluidAudio's AsrManager.transcribe() to throw `invalidAudioData`. Replacing non-finite values with 0.0 (silence) prevents the crash while preserving the rest of the transcription.

Output: Modified TranscriptionService.swift with one sanitization map in transcribeWithResampling().
</objective>

func transcribeWithResampling(_ samples: [Float], sampleRate: Double) async -> String? {

With:

```swift
func transcribeWithResampling(_ samples: [Float], sampleRate: Double) async -> String? {
    let samples16k = resampleTo16kHz(samples, fromSampleRate: sampleRate)
    // Sanitize: AVAudioEngine render-cycle drops can leave NaN/Inf in the buffer.
    // FluidAudio throws invalidAudioData on non-finite values — replace with silence.
    let sanitized = samples16k.map { $0.isFinite ? $0 : 0.0 }
    return await backend.transcribeWithResampling(sanitized, sampleRate: 16000.0)
}
```

No other changes to this file. Do not modify resampleTo16kHz, the backend, or AudioController.swift.
